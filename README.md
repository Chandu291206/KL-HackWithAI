# 🎓 EduCoach: Adaptive AI for Competitive Exam Prep
**Developed by Team Pixel Pioneers – KLH Hackathon**

EduCoach is an intelligent coaching agent designed to revolutionize JEE and NEET preparation by moving away from "one-size-fits-all" study plans. It utilizes reinforcement learning to adapt to a student's unique learning trajectory in real-time.

## ✨ Key Features
* **Adaptive Learning Logic:** Utilizes a **Multi-Armed Contextual Bandit** model to prioritize weak topics based on mock-test performance.
* **LLM Dynamic Scheduling:** Generates a personalized 7-day revision plan by analyzing topic scores and time-spent trends.
* **Contextual Library:** AI-generated theory summaries and curated YouTube recommendations color-coded by student mastery levels.
* **Smart Proctoring:** Includes tab-switch detection and a **Burnout Detection System** to suggest timely study breaks.
* **Deep Analytics:** Visualization through radar charts, heatmaps, and weekly AI-generated insight reports.

## 🛠️ Tech Stack
* **Language:** Python
* **Database:** SQLite
* **AI/ML:** Multi-Armed Contextual Bandit Algorithms, LLM APIs
* **Analytics:** Radar Charts, Streak Heatmaps
